PS CC 2013 安装

准备材料

PS CC 2013 压缩包（点击即可下载）

提取码：XYZF

百度网盘Windows客户端可以使用下载提速，这是方法

一、解压

用压缩工具解压压缩包，我这里使用7zip，或者WinRAR都可以，自行选择即可。

 

二、确认文件完整性

打开文件夹如图所示即可。

三、移动文件夹

此文件夹相当于安装包安装过后在 Program Files 或 Program Files(x86) 里的文件夹，如果理解可以自行选择存放位置，如不熟悉，按照步骤进行即可。

将文件夹全部移入C盘的 Program Files 目录下，如要索取管理员权限，点击继续即可。

 

四、安装

打开目录，并找到 QuickInstall(如果大小写有误忽略不计) ，点击安装按钮即可。

 

 

安装完成后即可在桌面看到

双击打开
